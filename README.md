# SFToolKey

SFToolKey is a desktop Python application for keyboard and mouse input automation.

It combines several useful tools in one interface:

- autoclicker
- random key presses
- random mouse movement
- simple macros
- settings profiles
- interface language switching: **Russian / English**

The app uses a modern dark UI and can be useful for anti-AFK tasks, input testing, simple automation scenarios, and custom macros.

---

## Features

### Main Module
- random key presses from a custom list
- configurable key hold time
- configurable delay between key presses
- random mouse movement
- configurable mouse movement interval
- configurable mouse movement strength
- separate hotkeys for start and stop

### Autoclicker
- mouse mode or keyboard mode
- mouse button selection:
  - left click
  - middle click
  - right click
- keyboard key selection using a bind box
- fixed interval mode
- random interval mode
- single, double, and triple click
- hold mouse button or keyboard key
- infinite mode
- run for a specific number of repeats
- click at current cursor position
- click at custom screen coordinates
- click counter

### Macros
- text-based macro editor
- line-by-line macro execution
- support for commands such as:
  - `WAIT`
  - `KEY`
  - `TEXT`
  - `CLICK`
  - `DOUBLECLICK`
  - `MOVE`
  - `HOLDKEY`
  - `HOLDMOUSE`
- macro start/stop hotkeys
- macro repeat modes:
  - once
  - specific count
  - infinite
- export macros to `.txt`

### Profiles
- save current settings as a profile
- load saved profiles
- delete profiles
- export profiles to JSON
- import profiles from JSON
- automatic local profile storage

### Extra
- interface language switching: `RU / EN`
- built-in action log
- `PANIC STOP` button to instantly stop everything
- hotkey capture using clickable bind fields
- protection against immediate hotkey triggering right after key assignment

# SFToolKey-ru

SFToolKey — это удобное desktop-приложение на Python для автоматизации ввода с клавиатуры и мыши.

Программа объединяет в одном окне:

- автокликер
- случайные нажатия клавиш
- случайное движение мыши
- простые макросы
- профили настроек
- переключение языка интерфейса: **Русский / English**

Приложение сделано с тёмным современным интерфейсом и подходит для задач вроде anti-AFK, тестов ввода, простых игровых сценариев и пользовательских макросов.

---

## Возможности

### Основное
- случайные нажатия клавиш из заданного списка
- настройка времени удержания клавиши
- настройка паузы между нажатиями
- случайное движение мыши
- настройка интервала движения мыши
- настройка силы движения мыши
- отдельные горячие клавиши для запуска и остановки

### Автокликер
- режим мыши или клавиатуры
- выбор кнопки мыши:
  - ЛКМ
  - СКМ
  - ПКМ
- выбор клавиши клавиатуры через удобную ячейку бинда
- фиксированный интервал
- случайный интервал
- одиночный, двойной и тройной клик
- удержание кнопки или клавиши
- бесконечный режим
- запуск на заданное количество повторов
- клик по текущей позиции курсора
- клик по заданным координатам
- счётчик выполненных нажатий

### Макросы
- текстовый редактор макросов
- выполнение команд по строкам
- поддержка простых действий:
  - `WAIT`
  - `KEY`
  - `TEXT`
  - `CLICK`
  - `DOUBLECLICK`
  - `MOVE`
  - `HOLDKEY`
  - `HOLDMOUSE`
- запуск макросов по горячим клавишам
- повтор макроса:
  - один раз
  - несколько раз
  - бесконечно
- сохранение макроса в `.txt`

### Профили
- сохранение текущих настроек в профиль
- загрузка профиля
- удаление профиля
- экспорт профилей в JSON
- импорт профилей из JSON
- автоматическое хранение профилей в файле

### Дополнительно
- переключение языка интерфейса: `RU / EN`
- лог действий внутри приложения
- кнопка `PANIC STOP` для мгновенной остановки всех модулей
- захват горячих клавиш через отдельную ячейку
- защита от мгновенного запуска сразу после назначения горячей клавиши
